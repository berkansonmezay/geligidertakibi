# Aile Bütçe Takip Sistemi (Family Budget Tracker)

Bu proje, aile bireylerinin gelir ve giderlerini takip edebileceği kapsamlı bir sistemidir.

## Proje Yapısı

- **/api**: Node.js, Express ve SQLite ile geliştirilmiş Backend API.
- **/web**: React ve Vite ile geliştirilmiş Responsive Web Uygulaması.
- **/mobile**: Flutter ile geliştirilmiş Mobil Uygulama (Android & iOS).

## Temel Özellikler
- Dinamik Yıl ve Periyot Yönetimi.
- Gelir ve Gider Kategorisi Özelleştirme (Ekleme/Düzenleme/Silme).
- Grafiksel Analizler (Pasta ve Çizgi Grafikleri).
- Cihazlar arası senkronizasyon (JWT Auth).
